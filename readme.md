# 🚀 PIXI.js Shoot 'em Up Demo (弾幕シューティングゲーム デモ)

このプロジェクトは、最新の**PixiJS v8**と**TypeScript**を使用して構築された、シンプルな弾幕系シューティングゲーム（STG/Shmup）のデモンストレーションです。

主に、**オブジェクトプール（Object Pooling）**、**コンポーネントベースの弾丸軌道システム**、および**依存性の注入（Dependency Injection）**を用いた、大規模なゲームでも保守しやすい構造の実現に焦点を当てています。

---

## 🎯 主な特徴と実装技術

* **コンポーネント化された弾丸システム (Complex Shot System)**:
    * `fireRateSpec` や `onDeathShot` といった高度な設定により、1つの弾丸が飛行中に子弾を発射したり、破壊時に爆発（二次弾）を生成したりするなど、複雑な弾幕パターンを容易に定義できます。
    * 弾丸は、直線 (`LINE`)、扇形 (`FAN`)、全方位 (`RING`) のパターンや、回転 (`ROTARY`) や波打ち (`WAVE`) の軌道を持つことができます。
    * 速度やスケール（サイズ）を動的に変化させる設定にも対応しています。
* **オブジェクトプール (`ObjectPool`)**:
    * 頻繁に生成・破棄されるエンティティ（弾丸、敵、爆発）に対してオブジェクトプールを適用し、ガベージコレクションの負荷を最小限に抑え、パフォーマンスを最適化しています。
* **クリーンなアーキテクチャ**:
    * `main.ts` が全てのコアモジュール (`InputManager`, `ScoreManager`, `EntityManager`) とエンティティを統合する役割を担うことで、各モジュールの責任が明確になっています。
* **イベント駆動型デザイン**:
    * `Player.SHOOT_EVENT` や `ScoreManager.SCORE_CHANGED_EVENT` などのカスタムイベントを通じてモジュール間が疎結合で連携します。

## 🕹️ 操作方法

プレイヤーの操作はシンプルで、キーボードの方向キーといくつかのショットキーを使用します。

| 操作 | キー | 説明 |
| :--- | :--- | :--- |
| **移動** | `←` / `→` (または `A`/`D`) | プレイヤーの左右移動 |
| **基本ショット** | `KeyA` | 基本の直線ショット |
| **扇形ショット** | `KeyS` | 扇形に広がるショット |
| **ロータリーショット** | `KeyD` | 発射のたびに角度が回転するショット |
| **ウェーブショット** | `KeyF` | 角度がサイン波のように左右に揺れるショット |
| **複合ショット** | `KeyQ` | 成長し、破壊時に子弾をばら撒く複雑な弾 |
| **飛行中/破壊時ショット** | `KeyR` | 飛行中に子弾を定期発射し、破壊時にも子弾を出す複合弾 |
